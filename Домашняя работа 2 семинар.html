Способы выделения текста в HTML

  • Чтобы выделить текст жирным используем тег "strong"
  • Для выделения курсивом необходимо возпользоваться тегом "em"
  • Для использования подчеркивания в тексте необходимо прописать тег "u"

Основные теги для работы

  1. Теги от "h1" до "h6" обозначают заголовки, где "h1" заголовок высшего ранга а "h6" соответсвенно низшего
  2. Каждый абзац обозначаем тегом "p"(paragraff)
  3. Для отделения текста чертой необходимо использовать тег "hr"

Не менее важным являеться способ размещения ссылки на страницы

Для этого нам необходимо прописать тег "a href="ссылка на сайт">"
